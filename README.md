# Navigateur et renommeur de PDFs

## Outils
- node v22.16.0+

## Démarrer le projet
```
# installer les dépendences (première fois uniquement)
npm install

# lancer le projet
npm start

# compiler en app electron
npm pack
```

## Structure du projet
```
pdf-renamer-app/
├── main.js
├── package.json
├── index.html
├── style.css
├── renderer.js
└── assets/ (dossier pour les icônes)
```
